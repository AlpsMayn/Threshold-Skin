Here are the multiple Customization options for Threshold.
* Collapsed Sidebar shrinks the left-hand sidebar
* Hide Account Name removes your Steam Account ID from the bottom left
* Collapsed Sidebar + Hide Account Name performs both of the above options
* _Default is the standard configuration

Other options are located in Misc. See the seperate README for more info

To install, copy the contents of the appropriate folder to the root of the skin (where config.ini is). Confirm you want to replace files, then restart steam.
